# CulinaShare-Global-Recipe-Nutrition-Explorer
🍴 CulinaShare | Global Recipe & Nutrition Explorer
📌 Overview

CulinaShare is a modern web application that enables users to explore, track, and analyze recipes from around the world. Designed with a focus on usability and performance, the platform transforms raw culinary data into an interactive and visually engaging experience.

It acts as a global food discovery and tracking system, allowing users to search recipes, filter cuisines, and save their favorite meals for future use.

🎯 Project Objective

The goal of this project is to demonstrate:

API integration using modern JavaScript

Efficient data manipulation using Higher-Order Functions

Responsive UI/UX design

Persistent data storage using browser technologies

📡 API Integration

API Used: TheMealDB API

API Link: https://www.themealdb.com/api.php

Data Fetching Method: ES6 fetch() with asynchronous handling

📦 Data Retrieved

Recipe Name

Ingredients & Measurements

Cooking Instructions

Meal Category (e.g., Vegan, Dessert)

Cuisine/Region (e.g., Italian, Indian)

Recipe Images

🚀 Core Features
🔍 Smart Search

Search recipes by name or ingredient

Implemented using .filter() for real-time results

🌍 Cultural Segmentation

Filter meals by:

Category (Vegan, Seafood, etc.)

Region (Italian, Mexican, etc.)

🔤 Alphabetical Sorting

Sort recipes alphabetically using .sort()

⭐ Personal Cookbook

Save favorite recipes

Uses Local Storage for persistence across sessions
